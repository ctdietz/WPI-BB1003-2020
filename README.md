# Virtual Environment

Activating the virtual environment:
If using an IDE such as VSCode, you can change the interpreter to the one in /VENV

Windows powershell:
In \WPI-BB1003-2020
'''
VENV\Scripts\Activate.ps1
'''

Windows CMD:
'''
VENV\Scripts\activate.bat
'''

Unix/Linux:
'''
source VENV/bin/activate
'''
